## Employee Performance Analytical Dashboard

This project is an interactive Power BI dashboard developed to analyze and track employee performance using KPIs and key business metrics.

## Tools & Technologies
- Power BI
- SQL Server
- Microsoft Excel

## Description
The dashboard integrates employee data from SQL Server and Excel to generate visual insights on:
- Salary vs Experience
- Role-wise performance
- Leadership and problem-solving ratings
- Professional knowledge by experience

Interactive filters allow users to analyze employee data dynamically across multiple dimensions.
